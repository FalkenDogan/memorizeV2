Quiz App User Guide

Introduction

Quiz App is an application that helps you easily create quizzes using Google Sheets. This guide explains step-by-step how to use it.

Step 1: Prepare Google Sheets

  • Create a new spreadsheet in Google Sheets.
  • Write your questions in Column A.
  • Write the correct answers in Column C.
  • Leave Column B just put the | (pipe) character.
  • Click the link for an example sheet here

Step 2: Get the Google Sheets Link

  • Open your Google Sheets document.
  • Click the "Share" button in the top right corner and select "Copy link".
  • Make sure the link contains the gid parameter (e.g., https://docs.google.com/spreadsheets/d/1aBcD.../edit?gid=1917887240#gid=1917887240).

Step 3: Use the Quiz App Application

  • Open the Quiz App application.
  • On the main page, paste the copied link into the "Insert Google Sheets Link" field.
  • Click the "Send" button.

Step 4: Quiz Settings

  • You will be redirected to the Question Selection page.
  • Enter the question range you want to include in the quiz in the "Start Question" and "End Question" fields.
  • Click the "Start Quiz" button.

Step 5: Taking the Quiz

  • On the Quiz page, questions will be displayed one by one.
  • Select the correct answer and click the "Submit" button.
  • Click the "Next" button to move to the next question.
  • At the end of the quiz, you can see your results.

Additional Features

  • Show Answer: You can see your incorrect answers and the correct answers.
  • Retry: You can restart the quiz.
  • Exit: You can return to the main page.

Important Notes

  • This programme does not access or process personal data in any way.
  • This program does not have a database. Questions are not saved in the system. All data is deleted when the program is closed.
  • Google Sheets links may generate different 'gid' numbers on computer browsers compared to smartphone and tablet applications.
  • If you are getting the link from the Google Drive or Google Sheets app on a smartphone or tablet, create the list only on the first tab of the Google Sheet and get the link.
  • If you are getting the link from a browser on a computer, you can create lists on each tab and use the links of the tabs separately.
